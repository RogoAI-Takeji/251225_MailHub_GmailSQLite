## Note

This software bundles copies of `tkinterweb` and `tkinterweb-tkhtml` to provide 
HTML rendering functionality without requiring users to install additional dependencies.

If you prefer to use your system-installed versions of these libraries, you can
remove the `lib/tkinterweb/` and `lib/tkinterweb_tkhtml/` directories from this project.
